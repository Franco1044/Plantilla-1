Herramientas

HTML

HTML (Lenguaje de Marcado de Hiperptexto) es el lenguaje estándar para estructurar el contenido de las páginas web, utilizando etiquetas para organizar textos, imágenes, videos y otros elementos, creando el esqueleto fundamental que los navegadores interpretan para mostrar la información.

CSS

CSS (Cascading Style Sheets) es el lenguaje que define la apariencia y el diseño visual de las páginas web, controlando aspectos como colores, fuentes, márgenes y diseño, separando la estructura (HTML) de la presentación para facilitar la gestión y crear sitios web atractivos y responsivos.

JavaScript

JavaScript (JS) es un lenguaje de programación fundamental para la web que añade interactividad y dinamismo a las páginas estáticas, trabajando con HTML y CSS para crear experiencias interactivas como animaciones.
